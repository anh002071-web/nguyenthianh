# nguyenthianh
medchain stellar
TÊN DỰ ÁN:
MedChain Stellar – Hệ thống quản lý hồ sơ bệnh án điện tử phi tập trung

VẤN ĐỀ (1 câu):
Bệnh nhân và bệnh viện đang gặp vấn đề về thất lạc, thiếu liên thông và rò rỉ dữ liệu hồ sơ bệnh án trong hệ thống lưu trữ truyền thống.

GIẢI PHÁP (1 câu):
dApp sử dụng Stellar để lưu hash hồ sơ bệnh án, cấp quyền truy cập bằng smart contract (Soroban) và cho phép chia sẻ dữ liệu y tế an toàn giữa các cơ sở.

TÍNH NĂNG STELLAR SỬ DỤNG:
[x] Chuyển XLM/USDC
[x] Token tùy chỉnh
[x] Soroban contract
[ ] DEX tích hợp
[x] Trustline
[x] Clawback/Tuân thủ

NGƯỜI DÙNG MỤC TIÊU:
Bệnh nhân, bác sĩ, bệnh viện, phòng khám và cơ quan quản lý y tế.

TÍNH NĂNG CỐT LÕI (MVP):
Một giao dịch cấp quyền truy cập hồ sơ bệnh án: bệnh nhân dùng ví ký giao dịch → smart contract cấp quyền cho bác sĩ → bác sĩ truy cập dữ liệu off-chain đã được xác thực bằng hash trên Stellar
